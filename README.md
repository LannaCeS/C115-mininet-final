# C115-mininet-final
Códigos referentes ao projeto final de Mininet da matéria de C115.

# Arquivos
trabalho-final-mininet_LannaC 1.py: código base do projeto, onde está a main e a classe que estabelece a topologia.
nodes 1.sh: arquivo  shell contendo comandos referentes à primeira parte da topologia da questão 2.
ping 1.sh: arquivo shell contendo comandos referentes à segunda parte da topologia da questão 2 (após reconexão via endereço MAC)
Documentação_Questões_1_e_2_mininet.pdf: arquivo pdf contendo a explicação detalhada sobre o funcionamento do projeto, seu passo a passo e resultados de execução.

# Execução
Para executar o projeto é necessário ter o Mininet instalado e funcional em sua máquina/máquina virtual.
